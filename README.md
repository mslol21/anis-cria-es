# Anis Criações - Personalizados que Vendem e Encantam

Projeto para gestão e apresentação da Anis Criações, especializada em canecas, camisetas, brindes e material gráfico personalizado.

## Tecnologias Utilizadas

Este projeto foi construído com:

- **Vite** - Build tool ultrarrápida
- **React** - Biblioteca para interfaces de usuário
- **TypeScript** - Tipagem estática para JavaScript
- **Tailwind CSS** - Framework CSS utilitário
- **Shadcn UI** - Componentes de interface reutilizáveis
- **Framer Motion** - Biblioteca para animações

## Como Rodar o Projeto Localmente

1. Clone o repositório
2. Instale as dependências:
   ```sh
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```sh
   npm run dev
   ```

## Estrutura do Projeto

- `src/components`: Componentes reutilizáveis da interface.
- `src/pages`: Páginas da aplicação.
- `src/hooks`: Hooks customizados do React.
- `src/lib`: Utilitários e configurações de terceiros.
